Project and model specifications

Project

In the list called “project” you can specify basic information about your project.

Excel screenshot example:

Specification in the excel:

Name of the column header

Description

Name

The name of the project

Description

The description of the project

Project nr.

The project number

Created

The creation time
Last Update The date and time of the last update
Project type The type of the project. The project type can be assigned to as "Building construction" or "Infrastructure construction".
Project kind The kind of the project. The project kind can be assigned to "Extension", "Finish", "New building", "Reconstruction", "Building alteration".
Building type Kind of use of the building such as Administrative, Residential, etc…
Status Status of the project. The project status can be assigned to as "in planning stage", "under construction", "in operation", "dismantling", "in alteration", "completed".
Location Location of the building

 

Model

In the list called “model” you can specify basic info about your project.

Excel screenshot example:

Specification in the excel:

Name of the column header

Description

Name

The name of the model

Description

The description of the model

Discipline

Discipline can be set as Undefined, Architecture, HVAC, Load-bearing structure, Terrain, Facility

LoD

LoD can be set as Undefined, Draft, Submission, Execution, Management
Status The date and time of the last update
Owner Type of the project. The project type can be assigned to as "Building construction" or "Infrastructure construction"
Revision number Kind of the project. The project kind can be assigned to "Extension", "Finish", "New building", "Reconstruction", "Building alteration"
Created The Creation time
Last Update The date and time of the last update
Source type Source type as one of direct, IFC, Sketchup, ...
Source application Source application as one of Automatic, Undefined, Allplan, Revit, ……
Source company
