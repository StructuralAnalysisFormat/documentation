Analytical Exchange Format (AEF)

Version 0.8

Welcome to the AEF website.

AEF is a global format for the exchange data from different analysis software via excel format in the structural engineering discipline.

AEF defines the description of the format of the data in excel and is not providing any excel files or data itself.

Go to Introduction or to Objects in the menu on the left side.
