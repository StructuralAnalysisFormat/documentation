StructuralSurfaceMemberOpening

Opening in surface member

Instances of StructuralSurfaceMemberOpening describe openings in planar members (StructuralSurfaceMember). The geometry of the opening object is defined in the same way as geometry of 2D members - may be planar or curved with an arbitrary number of edges.

Specification in the excel:

Name of the column header

Type of data

Value example or enum definition

Description

Name

String

O1

Human readable unique name of the object

2D Member

String

S13

The name of the StructuralSurfaceMember to which is the opening situated.

Nodes

String

N81; N263; N659; N660

All nodes that belong to surface member and define its geometric shape. The names of the nodes are separated by ; (semicolon) and space.

Edges

String

Line; Line; Line; Line

Defines shape of the curve between two next nodes. Supported strings are: Line; Bezier; Circular Arc; Parabolic arc; Spline. The names are separated by ; (semicolon) and space.
ID String - It is unique identifier of the object in BIMplus environment. It is automatically generated by BIMplus and user is not allowed to modify it.
