StructuralPointSupport

Support in node

StructuralPointSupport is representing support of the analytical model in node. The support is defined by six separate parameters. Each parameter defines the constraint in one direction: translation in X, Y, Z axis and rotation around the same axes. A nodal support is oriented in global co-ordinate system.
