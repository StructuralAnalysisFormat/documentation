Free line force

The Free line load is related to slabs. The load is not defined by the entity it acts on, but by a specific load border. Free loads are defined by means of "loading entities" that may overlap or affect one or more slabs.

This property defines what the load is caused by
Distribution

Enum

Uniform; Trapezoidal

The load may be either constant along its length or linearly variable (trapezoidal).
Direction Enum X; Y; Z Specifies the base direction of the load
Validity Enum All; MinusZ; ZIsZero; PlusZ

The validity of free load means which particular slabs are subject to the given load and the halfspace where the load acts.

All - Both entities located under and above the defined load are subject to the defined load.

MInusZ (-Z) - The load is supposed to act only in the space located along the negative half of Z-axis of the LCS.

ZIsZero (Z = 0) - Only the entities located in the XY-plane of the LCS are subject to the load in question.

PLusZ (+Z) - The load is supposed to act only in the space located along the positive half of Z-axis of the LCS.
Value 1 [kN/m] Double -150 Specifies the first size of the load in kiloNewtons per meter
Value 2 [kN/m] Double -180 Specifies the second size of the load in kiloNewtons per meter
Load case String LC1 The name of the load case to which the force belongs
Coordinate X [m] String 0.000; 2.050; 4.850; -2.000 The list of X coordinates of the nodes which define the geometry of the line free load. Each coordinate is separated by semicolon and space.
Coordinate Y [m] String 0.500; 1.050; 2.650; -1.500 The list of Y coordinates of the nodes which define the geometry of the line free load. Each coordinate is separated by semicolon and space.
Coordinate Z [m] String 0.000; 0.000; 0.000; 0.000 The list of Z coordinates of the nodes which define the geometry of the line free load. Each coordinate is separated by semicolon and space.
Segments String Line; Circular Arc List of the shape of each segment which is part of the curve, separated by semicolon if there are multiple
Coordinate system Enum Global; Local Defines co-ordinate system in which the load is applied
Location Enum Length; Projection Specifies whether the load is "put directly on an inclined 1D member" or whether the "projection on plan" is defined.
