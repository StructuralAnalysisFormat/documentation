Free Surface load

The Free line load is related to slabs. The load is not defined by the entity it acts on, but by a specific load border. Free loads are defined by means of "loading entities" that may overlap or affect one or more slabs.

Standard; Wind; Snow; Self Weight; Hoar Frost; Predefined; Plane Load; Water Pond; Water Pressure; Soil Pressure; Generated Water; Generated Soil

This property defines what the load is caused by
Direction Enum X; Y; Z Specifies the base direction of the load
Distribution Enum Uniform; Direction X; Direction Y; Direction XY

The load may be either constant across the slab or linearly variable
Validity Enum All; MinusZ; ZIsZero; PlusZ

The validity of free load means which particular slabs are subject to the given load and the halfspace where the load acts.

All - Both entities located under and above the defined load are subject to the defined load.

MInusZ (-Z) - The load is supposed to act only in the space located along the negative half of Z-axis of the LCS.

ZIsZero (Z = 0) - Only the entities located in the XY-plane of the LCS are subject to the load in question.

PLusZ (+Z) - The load is supposed to act only in the space located along the positive half of Z-axis of the LCS.
q [kN/m2] Double -10 Specifies the size of the load in kiloNewtons per square meter. This value is considered only if Distribution is set to Uniform
q1 [kN/m2] Double -5 Specifies the size of the load in kiloNewtons per square meter. This value is considered only if Distribution is set to Direction X or Direction Y or Direction XY. The Value is determined in the most distant points of surface load geometry in particular direction (X, Y).
q2 [kN/m2] Double -10 Specifies the size of the load in kiloNewtons per square meter. This value is considered only if Distribution is set to Direction X or Direction Y or Direction XY. The Value is determined in the most distant points of surface load geometry in particular direction (X, Y).
q3 [kN/m2] Double -5 Specifies the size of the load in kiloNewtons per square meter. This value is considered only if Distribution is set to Direction XY. The Value is determined in the most distant points of surface load geometry in particular direction.
Load case String LC5 The name of the load case to which the force belongs
Coordinate X [m] Double 0.000; 2.050; 4.850; -2.000 The list of X coordinates of the nodes which define the geometry of the surface free load. Each coordinate is separated by semicolon and space.
Coordinate Y [m] Double 0.500; 1.050; 2.650; -1.500 The list of Y coordinates of the nodes which define the geometry of the surface free load. Each coordinate is separated by semicolon and space.
Coordinate Z [m] Double 0.000; 0.000; 0.000; 0.000 The list of Z coordinates of the nodes which define the geometry of the surface free load. Each coordinate is separated by semicolon and space.
Edges String Line; Line; Line; Line Defines shape of the curve between two next nodes. Supported strings are: Line; Bezier; Circle arc; Parabolic arc; Spline. The names are separated by ; (semicolon) and space.
Coordinate system Enum Global; Local Defines co-ordinate system in which the load is applied
Location Enum Length; Projection Specifies whether the load is "put directly on an inclined 2D member" or whether the "projection on plan" is defined.
