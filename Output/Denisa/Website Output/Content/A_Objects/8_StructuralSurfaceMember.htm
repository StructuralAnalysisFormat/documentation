StructuralSurfaceMember

2D member (Plate, Wall…)

Instances of StructuralSurfaceMember describe face members, structural analysis idealizations of slabs, walls, shells. Surface members may be planar or curved with an arbitrary number of edges.

The geometry of 2D member is defined by nodes (StructuralPointConnection)and edges. The edges define shape of the curve between two next nodes.

MAT_1

The name reference to the existing, valid name of the StructuralMaterial object.

Shape

Enum

Flat; Curved

Specify the shape of the StructuralSurfaceMember in the sense of the planarity of the system plane.
Behaviour in Analysis Enum Isotropic; Otrhotropic; membrane; press only Isotropic: A normal isotropic slab with identical properties in all directions is used. Orthotropic: An orthotropic slab with different properties in two orthogonal directions is used. Membrane: Special membrane elements are used for the analysis of the slab. Press only: Special elements capable of resisting only compression stress are used for the analysis of the slab.
Layer String 1st floor Custom created layer. The layer can thus comprise entities that have something in common (e.g. one floor, columns of one floor, columns of the same length, etc.)
Thickness [mm] Double 200 The thickness of the structural surface member
System plane at Enum Bottom; Centre; Top Defines position of the system plane.
Eccentricity ez [mm] Double -125 The eccentricity of the slab may be input
Coordinate Z [m] Double 0,0 Coordinate of the vector of the LCS in Z direction
LCS Rotation [deg] Double 45.00 This value defines the rotation of local axes of the 2D member
Nodes String N81; N263; N659; N660 All nodes that belongs to surface member and defines its geometric shape. The names of the nodes are separated by ; (semicolon) and space.
Edges String Line; Line; Circular Arc; Line Defines shape of the curve between two next nodes. Supported strings are: Line; Bezier; Circular Arc; Parabolic arc; Spline. The names are separated by ; (semicolon) and space.
Colour String #7FFFFF00 Defines colour and transparency of the object. Colour is defined by Hex format #AARRGGBB. Transparency is controlled by the alpha channel AA.
ID String - It is unique identifier of the object in BIMplus environment. It is automatically generated by BIMplus and user is not allowed to modify it.
