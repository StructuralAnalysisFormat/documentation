StructuralCurveAction

Line force on the beam

The Line force load models load distributed over a 1D member or on a slab edge. It may be action along the whole 1D member or only on its part. It can be constant or trapezoidal, acting in three main directions X, Y, Z (global or local coordinate system).

Enum

on beam; on edge

Specifies on which type of object the force acts
Distribution Enum Uniform; Trapez The load may be either constant along the 1D member or linearly variable (trapezoidal).
Direction Enum X; Y; Z

Specifies the base direction of the load
Member String B11 The name of the curve member on which is the load applied. This value is considered only if Force action is set to “on beam”
2D Member String S1 The name of the StructuralSurfaceMember on which is load applied. This value is considered only if Force action is set to “on edge”
Internal edge String ES2 The name of the StructuralCurveEdge on which is load applied. This value is considered only if Force action is set to “on edge”
Edge Integer 1 Index of the edge of the the StructuralSurfaceMember on which is load applied or index of the segment of the StructuralCurveEdge on which is load applied
Value 1 [kN/m] Double -150 Specifies the first size of the load in kiloNewtons per meter
Value 2 [kN/m] Double -180 Specifies the second size of the load in kiloNewtons per meter
Load case String LC1 The name of the load case to which the force belongs
Coordinate system Enum Global; Local Defines co-ordinate system in which the load is applied
Location Enum Length; Projection Specifies whether the load is "put directly on an inclined 1D member" or whether the "projection on plan" is defined.
Coordinate definition Enum Absolute; Relative Selects the coordinate system that is used to define the length of the line load. Relative means without units. To define length of the load in meters input absolute
Extent Enum full; span The extent of force on the beam.
Start point [m] Double 0,3 Defines the position of the start point of the force in relative or absolute coordinates
End point [m] Double 0,65 Defines the position of the end point of the force in relative or absolute coordinates
Origin Enum From start; From end Specifies the origin of the coordinate system used for the definition of the length of the force
Eccentricity ey [mm] Double -150 Eccentricity of the force system line in Y direction of the local coordinate system
Eccentricity ez [mm] Double 75 Eccentricity of the force system line in Z direction of the local coordinate system
