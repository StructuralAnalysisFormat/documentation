StructuralEdgeConnection

Supports on 2D members edges

Object definition for a line support on a 2D member (StructuralSurfaceMember) edge. The support can be defined along the entire length of an edge or on its part only.

Human readable unique name of the support

Constraint of the support. The way the support acts in individual directions

Translation X

Enum

Free; Rigid; Flexible

Translation in X direction. Free - That is it imposes no constraint in the direction. Rigid - The connection in fully rigid in the specified direction. Flexible - The connection is flexible (elastic) in the specified direction.

Rx Enum Free; Rigid; Flexible Rotational stiffness around X axis

Ry Enum Free; Rigid; Flexible Rotational stiffness around Y axis

Rz Enum Free; Rigid; Flexible Rotational stiffness around Z axis
StiffnessFiz [MNm/rad/m] Double 50 The flexibility in rotation of the connection around local Z axis. Use this property only if the Rotational stiffness Rz is Flexible.
2D Member String S3 The name of the surface member to which is hinge related
Edge Integer 1 The index of edge of the surface member. The index starting with 1. The order is according to order of “edges” property at StructuralSurfaceMember.
Coordinate definition Enum absolute; relative Selects the coordinate system that is used to define the length of the hinge. Relative means without units. For define length of the hinges in meters input absolute

End point Double 0,75 Defines the position of the end point of the hinge in relative or absolute coordinates
Origin Enum From start; From end Specifies the origin of the coordinate system used for the definition of the length of the hinge
