Profile type

Enum

Timber; Geometric; Concrete; GeomThinWalled; SteelRolled

Define type of profile library:

Timber: Cross-section typical for wooden structures

Geometric: General geometric shapes

Concrete: Cross-section typical for concrete structures

GeomThinWalled: The thin-walled cross-section types

SteelRolled: Steel rolled profile types

Shape

Enum

Concrete_I_Section

Shape is required only if profile type is not SteelRolled. Complete list of supported shapes is attached in Annex.
Parameters [mm] String 50; 80; 500; 450 Parameters property is required only if profile type is not SteelRolled. Complete list of supported parameters is attached in Annex.
Rolled section String IPE180 This field is required only if profile type is SteelRolled. Defines name of the steel rolled profile in the globally common format.
Form code Enum 1 This field is required only if profile type is SteelRolled. The shape of the cross-section is uniquely identified by a so-called Formcode. The Formcode defines the shape, the parameters which describe the shape and in some cases also additional parameters like distance between bolt holes, unit warping coordinates etc.
Description ID of the profile Enum 2 This field is required only if profile type is SteelRolled. The description of the steel rolled cross-section.
